# الحساب الرسمي للتطبيق — Retweet

## بيانات الدخول

| الحقل | القيمة |
|--------|--------|
| **اسم المستخدم** | `@retweet` |
| **البريد** | `official@retweet.app` |
| **كلمة المرور** | `Retweet@Official2026!` |
| **معرّف الحساب** | `u_official_retweet` |

## التمييز في الواجهة

- شارة **بنفسجية** بجانب الاسم (ليست التوثيق الأزرق ولا شارة المؤسس الذهبية)
- إطار **«حساب التطبيق الرسمي»** في الملف الشخصي
- ملاحظة: `✦ حساب التطبيق الرسمي`
- منشور ترحيبي ثابت في الخلاصة

## إضافة الحساب على السيرفر (قاعدة `users.json`)

```bash
cd backend
node scripts/add-official-app-user.mjs
```

يمكن تعيين مسار البيانات:

```bash
set DATA_ROOT=D:\RetweetSocial
node scripts/add-official-app-user.mjs
```

## وضع التطوير المحلي

الحساب مضمّن تلقائياً في بذرة التطبيق (`store.tsx`) ويُعاد إدراجه عند تطبيع الحالة المحفوظة.

**غيّر كلمة المرور بعد أول استخدام في الإنتاج.**
